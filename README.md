# Let Me Help
信息与资源分享网站

# 目录结构
```
letmehelp/               <-- 根目录
|
+- config.py             <-- 配置文件，这个文件包含了应用需要的大多数配置变量
|
+- instance/config.py    <-- 配置文件，这个文件包含不应该出现在版本控制的配置变量。其中有类似调用密钥和数据库URI连接密码。
|
+- log/                  <-- 存放各个组件生成的日志
|
+- static/               <-- 存放静态文件
|
+- templates/            <-- 存放模板文件
|
+- apps/                 <-- 存放项目
|
+- spider/               <-- 存放爬虫项目
|  |
|  +- site/              <-- 存放全站抓取的爬虫代码
|  |
|  +- update/            <-- 全局更新爬虫代码
+- util/                 <-- 基础类以及通用的工具类和方法
```
